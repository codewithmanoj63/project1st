Logical Operators

  • Used to combine or modify boolean expressions and control program flow
  • Types :
    1. || (OR)
    2. && (AND)
    3. ! (NOT)

|| (OR || या)

  • Any one condition true then it will execute the code
  • EXAMPLE : 
                   
                    if (swi == "on" || swi == "off" || swi == "damage") {

                    if (swi == "on") {
                         console.log("Bulb On");
                    }

                    if (swi == "off") {
                         console.log("Bulb Off");
                    }

                    if (swi == "damage") {
                         console.log("Switch Damaged");
                    }

                    } else {
                         console.log("Please select on or off only");
                    }

&& (AND || और)

  • Check for the every codition and all should be true
  • Example : 
                   
                    if (username == "" || password == "") {
                         console.log("Username or password can not be empty");
                    } else if (username == "unscripted_surya" && password == "123456") {
                         console.log("Welcome to your home page");
                    } else {
                         console.log("Username or password is incorrect");
                    }
